# NetOps 分享时间

- 每周我们选定1~3个主持，
- 共同分享一些有趣的东西，
- 点燃探索的兴趣，碰撞思想的火花。
- 传播开源文化，普及GNU/Linux知识。

如果您希望主持某个主题或是讨论某个问题， 欢迎提PullRequest。
